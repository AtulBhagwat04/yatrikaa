# 🌍 Yatrikaa

> **A modern travel companion that helps users discover, plan, and explore destinations with ease.**

Yatrikaa is a full-stack travel guide application built using **Flutter**, **Node.js**, and **MongoDB Atlas**. It enables users to explore famous tourist destinations, discover nearby attractions, create personalized trip plans, save favorite places, and navigate seamlessly using Google Maps—all from one beautiful and easy-to-use application.

---

## ✨ Key Features

* 🏞️ **Explore Tourist Destinations:** Browse famous historical, religious, adventure, and nature destinations with detailed information.
* 🔍 **Smart Search:** Quickly find places by name, category, or location with an intuitive search experience.
* 📍 **Nearby Places:** Discover nearby tourist attractions, hotels, restaurants, and other essential places using your current location.
* 🗺️ **Google Maps Integration:** Get directions, calculate distances, and navigate directly to destinations.
* ❤️ **Favorites:** Save your favorite places and access them anytime from your profile.
* 🧳 **Trip Planner:** Organize trips by creating personalized itineraries and managing travel schedules.
* ☁️ **Cloud-Based Backend:** Securely stores user data, favorites, and trip information using MongoDB Atlas.
* 📸 **Beautiful Destination Gallery:** View high-quality images along with descriptions, timings, entry fees, and travel tips.

---

## 🛠️ Tech Stack & Architecture

Yatrikaa is built using modern full-stack development practices following a clean and scalable architecture.

**Core Technologies:**
* **Frontend:** Flutter, Dart
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **State Management:** BLoC
* **Authentication:** JWT
* **Cloud Storage:** Cloudinary
* **Maps & Location:** Google Maps API
* **Architecture:** REST API + Client-Server Architecture

## 🧠 Technical Highlights

### 📍 Location-Based Discovery
Yatrikaa uses the device's GPS along with Google Maps APIs to help users discover nearby attractions, restaurants, hotels, and navigate to destinations in real time.

### ☁️ Full-Stack Cloud Architecture
The application communicates with a Node.js REST API, while MongoDB Atlas securely manages user accounts, favorites, and trip data. Images are optimized and delivered through Cloudinary for faster loading.

## 🚀 Getting Started

### Prerequisites
* Flutter SDK (Latest Stable)
* Android Studio / VS Code
* Node.js
* MongoDB Atlas Account
* Google Maps API Key

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Yatrikaa.git
```

2. Navigate to the project folder:

```bash
cd Yatrikaa
```

3. Install Flutter dependencies:

```bash
flutter pub get
```

4. Start the backend:

```bash
cd backend
npm install
npm run dev
```

5. Run the Flutter application:

```bash
flutter run
```
